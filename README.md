# Angular World Map

This project is an Angular application that displays an interactive SVG world map.  
Selecting a country on the map displays country data retrieved from the World Bank API.

## Features
- Interactive SVG rendering
- API integration
- Dynamic map data display

## Technologies
- Angular
- TypeScript
- SVG
- REST APIs

## Purpose 
Built as part of a JavaScript programming project to demonstrate API integreation and front-end data visualization.
